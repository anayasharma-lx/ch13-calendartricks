Why it works — algebra
Let the number = 10a + b (a=tens, b=units) Reversed = 10b + a Difference = (10b + a) − (10a + b) = 9b − 9a = 9(b − a) → always ÷ 9 ✓ Sum = (10a + b) + (10b + a) = 11a + 11b = 11(a + b) → always ÷ 11 ✓
Both work for any 2-digit number with different digits. Try a few — it never fails!
